🧟 Zombie Zone

Zombie Zone é um jogo 2D desenvolvido em Python utilizando a biblioteca Pygame, criado como trabalho acadêmico para a faculdade.
O jogo consiste em sobreviver ao ataque de zumbis pelo maior tempo possível, acumulando pontos ao eliminar inimigos.

🎮 Sobre o Jogo

No Zombie Zone, o jogador controla um personagem que deve derrotar zumbis enquanto evita ser atingido.
O jogo possui:

Menu inicial
Sistema de vida
Sistema de pontuação (Score)
Sons e imagens
Tela de Game Over
Executável .exe gerado para Windows

🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

Python

Linguagem principal utilizada para o desenvolvimento do jogo, lógica, controle de personagens, inimigos, pontuação e sistema geral.

Pygame

Biblioteca Python utilizada para desenvolvimento de jogos 2D, responsável por:

Renderização de imagens
Sons
Detecção de teclado
Movimentação de sprites
Colisões
Game loop
Pyenv
